Sobre

A GZ Sistemas busca aprimorar cada dia mais suas tecnologias, de maneira a trazer para seus clientes e colaboradores maneiras de facilitar sua interação com os software com foco na evolução.

Com mais de 30 anos, atuando em territorio nacional, com uma imagem criada a partir da busca constante de evolução e melhoria

A GZ Sistemas trás mais ma melhoria para vocês, a API sem necessitar troca de software, apenas evoluindo para um ambiente mais aprimorado.

Visão da plataforma

Missão: Cada momento a nossa missão pode ser alterada sendo direcionada diretamente com a da cliente, como somos uma empresa de automação comercial, porém a ideia principal de nossa missão é LEVAR A MELHOR SOLUÇÃO PARA CADA CLIENTE

Valores: Com valores bem definidos como um bom relacionamento entre cliente e empresa, uma visão de mercado com intuito de aprimorar qualquer necessidade do cliente e também com a integridade e responsabilidade de cuidar de cada cliente.

Ideais: Com a idéia de se concentrar na efetiva melhoraria e automação de atividades sem a necessidade de mudanças bruscas, um dos ideais é a construção de maneiras para melhorar cada vez mais as funcionalidades dos softwares GZ Sistemas.

Conceitos

O conceito de api vem da sigla Application Programming Interface ou, em português, Interface de Programação de Aplicativos.

O objetivo dessa documentação é orientar o desenvolvedor sobre como integrar com a API GZ Sistemas, descrevendo as funcionalidades, os métodos que irão ser utilizados, listando informações.

O mecanismo de integração com a API GZ Sistemas é simples, de modo que apenas com conhecimentos intermediários em linguagens de programação JAVA

Arquitetura

Sobre

A GZ Sistemas consegue trazer uma tecnologia de ponta para os clientes, pensando na otimização da capacidade de cada cliente de pequeno a grande porte, para isso a recepção do desenvolvedor é simples, de maneira que não necessite ter níveis avançados de conhecimento para implementar a API GZ Sistemas.

A API GZ Sistemas é baseada em arquitetura em nuvem e API REST que permite a integração com sistemas de automação comercial de parceiros.


Segurança

Com intuito de levar a melhor segurança para cada cliente, da maneira que sempre consegue, você consegue manter sua estrutura e otimizando com a API GZ Sistemas.

Tendo a segurança na gerencia de estoque, na subida de movimentos e manipulação de pedidos. Toda suas informações em segurança e eficazes

Produto

Regra para geração do arquivo ESTOQUEMIX.TXT: o arquivo ESTOQUEMIX.TXT, seguirá a mesma estrutura do arquivo ESTOQUE.TXT, mais deverá ter todos os produtos juntamente com a informação da loja (campo 85 do manual e observação 33) ao qual o mesmo faz parte. Sendo assim, caso um produto exista por exemplo nas lojas 1,2 e 3, as informações deverão estar nas 3 linhas do arquivo juntamente com sua devida loja.
